Hardware
A computer that stays on 24/7. This is your AI's home.

Get a dedicated machine. Best: Mac mini. Also works: spare laptop, Linux box, or cloud VPS. Never use your personal daily computer.

Install Homebrew and prevent sleep. Open Terminal (Cmd+Space, type "terminal").

copy/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" sudo pmset -a sleep 0

Create a Gmail for the AI. Do it on your phone (easier verification). Log in on the machine. This is a throwaway identity, never your personal email.

OpenClaw
The software that runs your AI. Gives it tools, memory, and chat access.

Install and start OpenClaw.

copybrew install openclaw openclaw gateway install openclaw gateway start

Runs in background, survives reboots.

Connect Telegram. Follow the OpenClaw Telegram guide. Once connected, message it. It replies like a friend. Also works with Signal, Discord, Slack, iMessage.

Let it create accounts. Tell your AI (via Telegram) to sign up for GitHub, Vercel, Supabase, or whatever you need using its Gmail. Optional: give it a Privacy.com virtual card ($20/month limit).

Share your projects. GitHub: add as collaborator. Vercel: invite to team. Supabase: add to org. Google Docs: share with its Gmail.

You can stop here. You now have a working AI assistant on Telegram. Everything below makes it better.
Obsidian
See everything your AI writes, on every device, beautifully formatted. End-to-end encrypted sync.

Install Obsidian and open the workspace.

copybrew install --cask obsidian

Open Obsidian > "Open folder as vault" > press Cmd+Shift+G > type ~/.openclaw/workspace

Enable Obsidian Sync. Settings > Core plugins > Sync > on. Log in, create a remote vault. Add Code to Excluded folders. Keep defaults (Markdown only).

Don't enable "Sync all file types." Code repos will overwhelm your sync.

Keep Obsidian alive and connect devices. Save as ~/Library/LaunchAgents/md.obsidian.keepalive.plist:

<?xml version="1.0" encoding="UTF-8"?> <!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd"> <plist version="1.0"> <dict> <key>Label</key><string>md.obsidian.keepalive</string> <key>ProgramArguments</key> <array><string>open</string><string>-a</string><string>Obsidian</string></array> <key>RunAtLoad</key><true/> <key>StartInterval</key><integer>300</integer> </dict> </plist>
copylaunchctl load ~/Library/LaunchAgents/md.obsidian.keepalive.plist

Then install Obsidian on your phone/laptop, enable Sync, select the remote vault. No extra config on secondary devices.

Enable semantic search. Every note searchable by meaning, running locally.

copyopenclaw config set agents.defaults.memorySearch.provider local openclaw config set agents.defaults.memorySearch.fallback none

Edit ~/.openclaw/openclaw.json and add your folders under agents.defaults.memorySearch:

"extraPaths": ["Contacts", "Lists", "Memos", "Tasks"]

Restart the gateway. First run downloads a ~313MB model, then indexes automatically.

Don't skip this. Without it the AI only searches its own memory. With it, your entire vault is searchable by meaning.
Security
Keep the blast radius small. If everything leaks, you lose a throwaway Gmail and a $20 card.

Dedicated machine, separate accounts. The AI gets its own computer, Gmail, GitHub, Vercel, everything. Never share your personal accounts.

Gateway stays local. Listens on 127.0.0.1 only. Never expose to the internet. Use Tailscale (free) for remote access.

SOUL.md safety rules + scoped tokens. Top of SOUL.md: no reading outside workspace, no destructive commands, no emails or spending without approval. All API tokens: minimum permissions, short expiration.

Remote screen access. For CAPTCHAs and card fields (~5% of tasks). Mac: Screen Sharing. Anywhere: Tailscale + VNC.

Prompt injection is unsolved. Malicious content could trick your AI. The LLM provider sees your data. Don't feed it secrets. Skills can be malicious. Only install what you've reviewed.

Pro Tips
Not required. Just useful things we've learned.
Folders one level deep. Contacts, Lists, Memos, Tasks. Tell your agent your system and it sorts files for you.
Whiteboard.md at vault root. "W" sorts to the bottom, always easy to find. Scratchpad for quick thoughts and voice dumps.
All code in Code/. Excluded from Obsidian Sync. Git handles code, Obsidian handles notes.
Give it a real task immediately. Don't configure for a week. Ask it to build something now.
Update weekly.
copyopenclaw update
Tell your agent to read this page. Once OpenClaw + Telegram are set up, paste this URL into Telegram and say "follow this guide." It can do the Obsidian and search steps for you.
