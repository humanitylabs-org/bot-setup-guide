Part 1
The Machine
Get a computer that stays on 24/7

Best: Mac mini. Also works: spare laptop, Linux box, or cloud VPS. Must never fully sleep.

sudo pmset -a sleep 0
Install Homebrew

Open Terminal (Cmd+Space, type "terminal"). Paste this:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

It asks for your password. Letters won't show as you type. Normal.

Install OpenClaw

This runs your AI. Gives it tools, memory, files, and chat access.

brew install openclaw openclaw gateway install openclaw gateway start

Runs in the background, survives reboots.

Part 2
Communication
Connect Telegram

Follow the OpenClaw Telegram guide. Once connected, message it. It replies like a friend.

Create a Gmail for the AI

Make a new Gmail on your phone (easier for verification). Log into it on the AI's machine.

Never use your personal Gmail. This is a throwaway identity.
Let it create accounts

Tell your AI to sign up for GitHub, Vercel, Supabase, or whatever you need. Optional: Privacy.com virtual card ($20 limit) for purchases.

Part 3
Knowledge Layer
Install Obsidian

See everything your AI writes, beautifully formatted, on any device.

brew install --cask obsidian

Open it. Click "Open folder as vault." Press Cmd+Shift+G and type:

~/.openclaw/workspace
Enable Obsidian Sync

$4/month, end-to-end encrypted. Settings > Core plugins > Sync > on. Create a remote vault. Add Code to Excluded folders.

Don't enable "Sync all file types." Markdown-only is correct.
Keep Obsidian running forever

Save as ~/Library/LaunchAgents/md.obsidian.keepalive.plist:

<?xml version="1.0" encoding="UTF-8"?> <!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd"> <plist version="1.0"> <dict> <key>Label</key><string>md.obsidian.keepalive</string> <key>ProgramArguments</key> <array><string>open</string><string>-a</string><string>Obsidian</string></array> <key>RunAtLoad</key><true/> <key>StartInterval</key><integer>300</integer> </dict> </plist>
launchctl load ~/Library/LaunchAgents/md.obsidian.keepalive.plist
Connect your other devices

Install Obsidian on phone/laptop. Create vault, enable Sync, select the remote vault. No extra config needed.

Enable semantic search

Search every note by meaning, not keywords. Runs locally, zero data sent externally.

openclaw config set agents.defaults.memorySearch.provider local openclaw config set agents.defaults.memorySearch.fallback none

Edit ~/.openclaw/openclaw.json, add your folders:

"extraPaths": ["Contacts", "Lists", "Memos", "Tasks"]

Restart the gateway. First run downloads a ~313MB model.

Don't skip this. Without it, the AI only searches its own memory. With it, it searches your entire vault.
Part 4
Security
Use a dedicated machine

Never run on your personal computer. The AI gets its own machine, its own user account.

Keep the gateway local

Gateway listens on 127.0.0.1 only. Never expose to the internet. Use Tailscale (free) for remote access.

Write safety rules in SOUL.md

No reading outside workspace. No destructive commands. No emails or spending without approval.

Use scoped tokens

Minimum permissions. Short expiration. If the AI's identity leaks, worst case = throwaway Gmail + $20 card.

Set up remote screen access

For CAPTCHAs and card fields (~5% of the time). Mac: Screen Sharing. Anywhere: Tailscale + VNC.

Part 5
Go
Share your projects

GitHub: add as collaborator. Vercel: invite to team. Supabase: add to org. Docs: share with its Gmail.

Give it a real task

Don't configure for a week. Ask it to build something now. You learn by doing.

Update regularly
openclaw update

Run weekly.

Bonus
Pro Tips
Pro Tip

Keep folders one level deep. Contacts, Lists, Memos, Tasks. Tell your agent how you organize and it will sort files for you.

Pro Tip

Create a Whiteboard.md. At the vault root. "W" sorts to the bottom = always easy to find. Use as a scratchpad.

Pro Tip

All code in Code/. Excluded from Obsidian Sync. Git handles code, Obsidian handles notes.

Important
Known Risks

Prompt injection is unsolved. Malicious content could trick your AI. That's why it gets throwaway accounts.

The LLM provider sees your data. Don't feed it truly secret information.

Skills can be malicious. Only install skills you've reviewed.
